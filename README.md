# DSA Python Programs

This repository contains beginner Python programs created while learning programming and data structures.

## Programs

### Traffic Signal Simulation
A simple Python program that simulates traffic light behavior.

File: `traffic_signal.py`

### Random Number Generator
Generates random numbers using Python's random module.

File: `random_number.py`

### Stack Operations
A program which inserts, deletes, and displays the element in the stack.

File: `Stack.py`

### Tree Traversal
A program which gives the inorder, preorder, postorder of the binary tree

File: `Tree Traversal.py`
